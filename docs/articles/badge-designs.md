# Badge Designs

Seven badge designs are included, covering three visual personalities and two physical sizes each.

All designs are dark-and-bold by default and include an inline SVG QR code or barcode (generated by the `{{qrCode}}` / `{{barCode}}` Handlebars helpers — no external service required).

---

## Sizes

| Size | Dimensions | Use case |
|---|---|---|
| **A6** | 105 × 148 mm | Lanyard badge, conference name tag |
| **Credit-card (CC)** | 85.6 × 54 mm | Wallet card, compact event pass |

## Creating a custom design

1. Copy an existing template pair, e.g. `badge-pulse-a6.html` + `badge-pulse-a6.css`
2. Create a companion JSON file referencing your new files via `htmlPath`/`cssPath`
3. Add the JSON to the `templates/` directory
4. The file-mode worker picks it up automatically on next run; in Kafka mode, send a `DocumentRenderRequest` with the template embedded inline or reference it from the host filesystem
